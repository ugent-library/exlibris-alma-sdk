# exlibris-alma-sdk

ExLibris Alma SDK for JavaScript

## Installation

```bash
bun add exlibris-alma-sdk
```

## Usage

```typescript
import { greet } from "exlibris-alma-sdk";

console.log(greet("World")); // Hello, World!
```
